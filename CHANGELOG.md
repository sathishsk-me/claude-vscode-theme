# Changelog

All notable changes to the **Claude Code Theme** extension will be documented in this file.

## [1.0.0] - 2025-05-23

### Added

- **Claude Code** — Primary dark theme with deep backgrounds and balanced contrast
- **Claude Code Warm** — Warmer variant with amber and orange tones
- Full syntax highlighting support for 20+ languages
- Optimized UI colors for editor, sidebar, terminal, status bar, and more
- Support for bracket pair colorization
- Git decoration colors
- Integrated terminal color scheme
- Sample files for theme testing (Java, JavaScript, CSS, HTML, Python, TypeScript/TSX, Markdown, XML, JSON)

### Technical Details

- Minimum VS Code version: 1.80.0
- Two theme variants included
- MIT licensed

## [2.0.0] - 2025-05-23

### Added

- **Claude Code Light** — Primary light theme with soft backgrounds and balanced contrast
- **Claude Code Light Warm** — Warmer variant with cream and orange tones

## [3.0.0] - 2025-05-25

### Added

- Update code syntax highlighting for better readability and contrast
- Updated title bar and menu colors for better visual separation and consistency across themes

## [Unreleased]

### Planned

- Additional theme variants (e.g., high contrast, pastel)
- Semantic token color support improvements
- More language-specific optimizations
